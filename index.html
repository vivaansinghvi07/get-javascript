Get Javascript!

Based on the popular game 2048, the objective of this game is to combine various programming languages to obtain JavaScript! The game mechanically deviates a little from the original, but that just adds more personality to it! (I am too lazy to code it to match exactly)

You can move with the arrow keys or WASD, but be careful: making any move (whether possible or not) adds a version of C to the board!
